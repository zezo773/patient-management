# patientservice package

This directory contains the main Java package for the Patient Service module.

## Purpose
- Hosts the core Spring Boot application code for patient-related functionality.
- Organizes source files by responsibility using sub-packages.

## Common sub-packages
- `controller/` – REST endpoints
- `service/` – business logic
- `repository/` – data access layer
- `model/` – domain entities and DTOs
- `config/` – configuration classes
